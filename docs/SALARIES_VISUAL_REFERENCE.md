## Formula Reference

### Employee Row Formulas (Example: Row 4)

```
Column F (Working days after deductions):
=D4-E4

Column G (Weekly salary without deductions):
=C4*D4

Column H (Weekly salary with deductions):
=C4*F4

Column J (Final salary after advances):
=H4-I4
```

### Week Total Formula (Example: Row 9)

```
Column J (Sum of all final salaries for the week):
=SUM(J4:J8)
```

### Grand Total Formula (Final Row)

```
Column J (Sum of all week totals):
=J9+J18+J27+J36+J45
(Adds up all week total cells)
```

## Cell Merging

### Week Header (Row 2, 12, 22, etc.)
```
Merged: B2:J2
Content: "رواتب الموظفين عن شهر ديسمبر - الأسبوع الأول"
```

### Week Total (Row 9, 18, 27, etc.)
```
Merged: B9:I9
Content: "إجمالي الرواتب للأسبوع"
Cell J9: =SUM(J4:J8)
```

### Grand Total (Final Row)
```
Merged: B:I
Content: "إجمالي الرواتب للشهر بالكامل"
Cell J: =J9+J18+J27+J36+J45
```

## Editable vs Calculated

### ✏️ Editable (User can modify)
- Column E (الخصومات) - Deductions
- Column I (سلف الموظف) - Advances

### 🔒 Calculated (Auto-updates)
- Column F (عدد أيام العمل بعد الخصومات) - =D-E
- Column G (الراتب الأسبوعي) - =C*D
- Column H (الراتب الأسبوعي بعد الخصم) - =C*F
- Column J (نهائي الراتب) - =H-I
- Week Totals - =SUM(J:J)
- Grand Total - =Sum of week totals

### 📊 From Attendance (Read-only)
- Column D (عدد أيام العمل) - Read from attendance file

### 📝 From Employee Data (Read-only)
- Column B (اسم الموظف) - Employee name
- Column C (يومية الموظف) - Daily rate
